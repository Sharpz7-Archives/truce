Go to the Releases Page to download the latest version of the software.

## Libraries
- [Ardiuno Threads](https://github.com/ivanseidel/ArduinoThread)

## Output

Digital Output Pins 2-4 are used.

|  State  | Pin 2 | Pin 3 | Pin 4 |
|---------|-------|-------|-------|
| RELAXED | 0     | 0     | 1     |
| GRIPPED | 0     | 1     | 0     |
| NONE    | 0     | 0     | 0     |
